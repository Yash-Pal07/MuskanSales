# Muskan Sales Website (React)

Clean React + Vite project for Muskan Sales service showcase website.

## Run locally

```bash
npm install
npm run dev
```

## Build production files

```bash
npm run build
```

## Project structure

- `src/App.jsx` - main page layout
- `src/components/` - reusable UI sections
- `src/data/siteData.js` - services, brands, and contact details (edit here first)
- `src/styles.css` - centralized UI styles

## Contact update

To update phone/WhatsApp/email, edit `CONTACT` inside `src/data/siteData.js`.
